# Message Board REST API

A small message board project built using **Node.js, Express and MongoDB**.  
Users can create, edit and delete messages with timestamps.

## Features

- Add a new message (sender, receiver, message body).
- Edit and delete existing messages.
- Shows date and time for each message.
- RESTful routes using Express.

## Tech Stack

- Node.js
- Express
- MongoDB / Mongoose
- EJS / HTML / CSS (for frontend cards)
